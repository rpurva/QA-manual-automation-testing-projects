# QA Automation Project using Selenium and Python

## Project Overview
This project automates key e-commerce user workflows using Selenium WebDriver for functional testing.

## Application Under Test
https://www.saucedemo.com/

## Test Scenarios Automated
- Login functionality (valid user)
- Add product to cart
- Cart verification

## Tools Used
- Python
- Selenium WebDriver
- Chrome Browser

## How to Run

1. Install dependencies:
   
   pip install -r requirements.txt

2. Run test scripts:
    python tests/login_test.py
   
    python tests/addtocart_test.py
