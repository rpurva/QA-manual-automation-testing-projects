# Job Application Form Testing Project

##  Overview
This project demonstrates manual testing of a simple job application form using test cases and bug reporting.

---

##  Tools Used
- HTML (Form)
- Excel (Test Cases & Bug Reports)
- Manual Testing
- GitHub

---

##  What I Did
- Created test cases for form validation and functionality  
- Executed test cases on the form  
- Reported bugs found during testing  
- Documented results with screenshots  

---

##  Bugs Found
- Phone field accepts alphabets  
- No success message after submission  
- File upload has no restriction  
- Dropdown is not mandatory  

---

## Files
- `testform.html` – Job application form  
- `testcases.xlsx` – Test scenarios(sheet 1) and bug reports (sheet2)
- `screenshots/` – Evidence of bugs
